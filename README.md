# MUST Resource Platform v4.0

資源教室公告管理平台。

## v4.0 新功能
- AI 公告助手
- AI 正式版、學生白話版、LINE 版、500字版、Emoji 版
- 無 API Key 時可自動產生 ChatGPT 提示詞
- 有 OpenAI API Key 時可直接在後台產生文字
- 修正附件中心排版錯亂
- 附件中心依公告分組

## 成本提醒
- GitHub Pages：免費
- Firebase Authentication / Firestore 小量使用：免費額度內
- GitHub Token：免費
- AI 若使用 OpenAI API Key，會依 OpenAI API 使用量計費
- 若不設定 OpenAI API Key，可使用「複製提示詞到 ChatGPT」模式，零 API 成本

## Token
GitHub Token 不需要重新貼，除非你清除瀏覽器資料或 Token 過期。
