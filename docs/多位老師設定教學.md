# 多位老師設定教學

## 第一步：上傳 v4.1

照原本方式覆蓋上傳 GitHub。

## 第二步：更新 Firestore Rules

Firebase → Firestore Database → Rules

把 `firebase/firestore.rules` 的內容貼上，按發布。

## 第三步：新增老師

用最高管理員帳號登入：

master004400@gmail.com

後台 → 系統設定 → 管理員老師名單

輸入老師 Google Email，例如：

teacher@example.edu.tw

按「新增老師」。

## 注意事項

1. 新增的老師必須使用該 Email 的 Google 帳號登入。
2. 建議使用學校信箱或固定 Google 帳號。
3. 不要新增學生 Email。
4. master004400@gmail.com 是預設最高管理員，即使名單被刪也仍可登入。
