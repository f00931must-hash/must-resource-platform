# v1.0 Beta 測試流程

## 1. 上傳檔案
解壓縮 v1.0 ZIP，將所有內容覆蓋上傳到 GitHub Repository。

## 2. 等 GitHub Pages 部署
右側 Deployments 顯示綠色 github-pages 後，即可測試。

## 3. 前台網址
https://f00931must-hash.github.io/must-resource-platform/

或

https://f00931must-hash.github.io/must-resource-platform/frontend/

## 4. 後台網址
https://f00931must-hash.github.io/must-resource-platform/admin/

## 5. Firebase Authentication 必須確認
Firebase → Authentication → Settings → Authorized domains

必須有：

f00931must-hash.github.io

## 6. 測試新增公告
登入後台後：
- 點公告管理
- 輸入標題
- 選分類
- 輸入內容
- 儲存
- 回前台確認是否出現

## 7. 圖片與附件
v1.0 Beta 先使用網址方式：
- 圖片網址每行一個
- 附件格式：名稱|網址

v1.1 會改成更完整的圖片中心與附件中心流程。
